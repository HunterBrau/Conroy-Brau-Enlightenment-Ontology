# Current Scope

The current project phase is deliberately narrow: produce clear conference
claims from the reproducible Wikidata/Wikipedia evidence already in the
repository.

## Active Analytical Spine

`global_writers` is the active corpus:

- Wikidata humans;
- born 1675-1775;
- occupation `writer` or subclass of writer;
- normalized to one row per Wikidata QID.

The legacy `french_seed` files are retained for provenance only.

## Active Comparison Units

France, Germany, British/British imperial contexts, and China/Qing contexts
are all derived from `global_writers` through the same reviewed
political-entity crosswalk.

This keeps France from being structurally different from the other comparison
groups.

## Active Evidence Families

- country of citizenship;
- birth, death, residence, and work-location place context;
- native, spoken/written, and writing-language evidence;
- Wikipedia language-edition sitelinks;
- Wikidata label and description coverage;
- gender;
- occupation buckets;
- data-friction metrics.

## Not Planned For This Phase

- BnF;
- other external authority systems;
- broadening beyond writer/subclass occupations;
- changing the identity spine away from Wikidata QIDs.

## Current Deliverable

The core findings packet is generated by:

```powershell
python scripts/analysis/08_build_core_findings_packet.py
```

Primary output:

- `docs/core_findings_packet.md`

```powershell
python scripts/analysis/09_build_insight_mining_packet.py
python scripts/visuals/02_build_insight_figures.py
```
