# Reference Tables

This folder stores small, reviewable crosswalks used by the analysis scripts.
They are not raw source exports. They are project-curated interpretation
layers, so collaborators should be able to inspect and revise them directly.

Current reference tables:

- `political_entity_affiliation_crosswalk_seed.csv`
- `occupation_bucket_crosswalk_seed.csv`

The occupation bucket crosswalk is generated by:

```powershell
python scripts/analysis/06_build_occupation_bucket_tables.py
```

It uses Wikidata occupation QIDs as stable identifiers and assigns suggested
granular buckets. Rows marked `needs_review` should be treated as open
curation work, not final categories.
